# Skill Groups

These folders organize the installed skills by working category without moving
the original skill directories.

Original installs remain in `.agents/skills/` so Codex skill discovery keeps
working. Each item in these folders is a symlink back to its source skill.

## Groups

- `frontend-design`: visual design, UI patterns, interaction design, brand, and
  frontend polish skills.
- `accessibility-inclusive`: keyboard, responsive, touch, assistive technology,
  and inclusive design skills.
- `research-product`: PM research, product strategy, user research, and journey
  mapping skills.
- `design-systems`: tokens, components, pattern libraries, and design system
  governance.
- `critique-audit`: review, QA, heuristic, compliance, and audit skills.
- `content-handoff`: writing, documentation, presentations, and handoff skills.
- `engineering-platform`: deployment, platform, framework, and implementation
  workflow skills.
- `misc-product-tools`: uncategorized product and UX utility skills.
